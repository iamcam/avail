##SublimeText
First, install Package Control: https://sublime.wbond.net
There are a couple of SublimeText configuration files in the repo, but there are a few packages you should install:
* Browser Refresh
* Markdown Preview
* JSON Reindent
* PrettyJSON
* SublimeCodeIntel
* SyncedSidebar
* SCSS
* HTML5
